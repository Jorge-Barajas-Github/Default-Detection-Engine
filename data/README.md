# \# Data

# 

# This project uses a structured financial dataset for predicting credit card default risk.

# 

# \---

# 

# \## Dataset Source

# 

# The dataset used in this project is publicly available on Kaggle:

# 

# Yeh, I. C., \& Lien, C. H. (2009). \*Default Payments of Credit Card Clients in Taiwan\* \[Data set]. Kaggle.  

# https://www.kaggle.com/datasets/uciml/default-of-credit-card-clients-dataset

# 

# \---

# 

# \## Data Description

# 

# The dataset contains information on 30,000 credit card clients and includes:

# 

# \- Demographic features (e.g., age, education, marital status)

# \- Credit limit information

# \- Historical repayment behavior (PAY\_0 to PAY\_6)

# \- Billing amounts (BILL\_AMT1 to BILL\_AMT6)

# \- Payment amounts (PAY\_AMT1 to PAY\_AMT6)

# 

# The target variable is:

# 

# \- `default.payment.next.month`  

# &#x20; - 1 = default  

# &#x20; - 0 = no default  

# 

# \---
